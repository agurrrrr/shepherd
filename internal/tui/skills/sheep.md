# /sheep - 양 관리

양 목록 조회 및 관리를 수행합니다.

## 사용법
```
/sheep [명령] [이름]
```

## 명령
- `/sheep` 또는 `/sheep list` - 양 목록 조회
- `/sheep spawn [이름]` - 새 양 생성 (이름 생략시 자동 생성)
- `/sheep delete <이름>` - 양 삭제

## 예시
- `/sheep` - 양 목록 조회
- `/sheep spawn` - 새 양 자동 이름으로 생성
- `/sheep spawn 복실이` - 복실이라는 양 생성
- `/sheep delete 양동이` - 양동이 삭제

## 실행
shepherd sheep 명령을 실행하세요:
- shepherd sheep list
- shepherd sheep spawn [이름]
- shepherd sheep delete <이름>

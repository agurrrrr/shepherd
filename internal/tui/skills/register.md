# /register - 프로젝트 등록

새 프로젝트를 등록합니다.

## 사용법
```
/register <경로> [이름]
```

## 예시
- `/register ~/code/myproject` - myproject 이름으로 등록
- `/register ~/code/myproject my-app` - my-app 이름으로 등록
- `/register .` - 현재 디렉토리 등록

## 실행
shepherd project add 명령을 실행하여 프로젝트를 등록하세요.

프로젝트 등록 후 자동으로 사용 가능한 양을 배정합니다.
배정 가능한 양이 없으면 새로 생성합니다.
